# Process list

Display opened processes as a graph.

Build with `Electron`, `React` and `Typescript`.

<p align="center">
  <img src="./screenshots/screen.png" alt="Screenshot"
       width="70%">
</p>


## Usage
Download the latest version npm and then run:
```
$ npm i
$ npm run build
$ npm start
```
